# Layout

There is currently a `landscape` and a `square` layout you can choose from to define the aspect ratio of your video.

<br />
<br />
<div
  style={{
    display: "flex",
    justifyContent: "space-around",
    gap: 10,
    alignItems: "center",
    maxWidth: "72rem",
  }}
>
  {/* <Layout type={"portrait"} aspectWidth={9} aspectHeight={16} /> */}
  <Layout type={"landscape"} aspectWidth={16} aspectHeight={9} />
  <Layout type={"square"} aspectWidth={1} aspectHeight={1} />
</div>
<br />
<br />

To change the layout, change the `canvasLayout` property in the right sidebar.
